# Built-in Mods

Symposium ships with three built-in mods that enhance your agent's capabilities for Rust development.

- **[Sparkle](./sparkle.md)** - AI collaboration framework that learns your working patterns
- **[Ferris](./ferris.md)** - Rust crate source inspection
- **[Cargo](./cargo.md)** - Compressed cargo command output

See [Using Symposium](./symposium.md) for how to configure mods.
